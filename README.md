# Real vs. Semi-Simulated: Rethinking Evaluation for
Treatment Effect Estimation

Conda environment:

`conda env create -f environment_real_vs_semisim.yml`

`conda activate real_vs_semisim`

Run configs (from this folder):

`python run.py --config config_metalearn.json`

`python run.py --config config_non_metalearners.json`

`python run.py --config config_criteo_st.json`
